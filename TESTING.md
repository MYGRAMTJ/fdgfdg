# 🧪 Тестирование и расширенное использование

## Тестовые сценарии

### Тест 1: Инициализация системы
```
Шаги:
  1. Откройте index.html в браузере
  2. Дождитесь загрузки страницы
  
Ожидаемые результаты:
  ✓ Интерфейс загружен
  ✓ Текст статуса: "Готово к старту"
  ✓ Кнопка "▶️ Запустить" активна
  ✓ Кнопка "⏹️ Остановить" неактивна
  ✓ Индикатор статуса красный
```

### Тест 2: Запуск камеры
```
Шаги:
  1. Нажмите "▶️ Запустить"
  2. Разрешите доступ к камере
  3. Дождитесь инициализации
  
Ожидаемые результаты:
  ✓ Видео из камеры отображается
  ✓ Текст статуса: "Камера активна"
  ✓ Индикатор статуса становится зеленым
  ✓ Кнопка "▶️ Запустить" неактивна
  ✓ Кнопка "⏹️ Остановить" активна
```

### Тест 3: Обнаружение руки
```
Шаги:
  1. Поднесите руку перед камеру
  2. Откройте все пальцы
  3. Медленно двигайте рукой
  
Ожидаемые результаты:
  ✓ На видео видно скелет руки (точки и линии)
  ✓ Поле "Видимые руки" показывает "1"
  ✓ Поле "Уверенность" показывает > 50%
  ✓ Виртуальный курсор появляется
  ✓ Курсор следует за указательным пальцем
```

### Тест 4: Управление курсором
```
Шаги:
  1. Откройте руку (все пальцы вверх)
  2. Двигайте рукой в разные стороны
  3. Проверьте плавность движения
  
Ожидаемые результаты:
  ✓ Виртуальный курсор следует за пальцем
  ✓ Движение плавное (без рывков)
  ✓ Координаты обновляются в поле "Позиция курсора"
  ✓ Статус показывает "Открытая ладонь" или "Раскрытая рука"
```

### Тест 5: Жест "Кулак" (Клик)
```
Шаги:
  1. Откройте руку перед камерой
  2. Медленно сожмите все пальцы в кулак
  3. Подержите кулак 1 секунду
  4. Откройте руку
  
Ожидаемые результаты:
  ✓ Статус меняется на "Кулак (Клик)"
  ✓ Появляется анимация 💥 в месте курсора
  ✓ В истории жестов появляется 👊
  ✓ Если был наведен на кнопку - кнопка нажимается
```

### Тест 6: История жестов
```
Шаги:
  1. Выполните несколько разных жестов подряд
  2. Сожмите кулак → откройте руку → покажите V → откройте руку
  3. Посмотрите на "Список распознаанных жестов"
  
Ожидаемые результаты:
  ✓ В истории отображаются последние 5 жестов
  ✓ Новые жесты добавляются слева
  ✓ Последний жест - крайний справа
  ✓ Каждому жесту соответствует правильный emoji
```

### Тест 7: Две руки
```
Шаги:
  1. Поднесите очень две руки перед камеру
  2. Держите обе открытыми
  3. Наблюдайте за изменениями
  
Ожидаемые результаты:
  ✓ Поле "Видимые руки" показывает "2"
  ✓ видны два скелета рук
  ✓ Два курсора следуют за пальцами
  ✓ Последняя обнаруженная рука управляет курсором
```

### Тест 8: Регулировка чувствительности
```
Шаги:
  1. Установить "Уверенность обнаружения" на 30%
  2. Попробуйте показать полурасткытую руку
  3. Установить на 90%
  4. Попробуйте показать ту же руку
  
Ожидаемые результаты:
  ✓ При 30% распознаёт почти любую форму руки
  ✓ При 90% требует четкой, уверенной позиции
  ✓ При 90% меньше ложных срабатываний
```

### Тест 9: Сглаживание
```
Шаги:
  1. Установить сглаживание на 0%
  2. Быстро двигайте рукой
  3. Установить на 100%
  4. Быстро двигайте рукой снова
  
Ожидаемые результаты:
  ✓ При 0% - курсор "дергается", точен но шумно
  ✓ При 100% - курсор очень плавный, но отстает
  ✓ Оптимум около 60-70%
```

### Тест 10: Сброс
```
Шаги:
  1. Выполните несколько жестов
  2. Нажмите кнопку "🔄 Сброс"
  3. Посмотрите на историю жестов
  
Ожидаемые результаты:
  ✓ История жестов очищена
  ✓ Текущий жест: "Нет"
  ✓ Список жестов пуст
  ✓ Система продолжает работать
```

## Что нужно тестировать регулярно

### Производительность
- [ ] FPS видео плавный
- [ ] Курсор не дергается при движении
- [ ] Нет задержки в распознавании
- [ ] Память браузера не растет бесконечно

### Точность
- [ ] Жест "Кулак" распознается при полном зажатии
- [ ] Ладонь распознается при открытых пальцах
- [ ] V-знак распознается при двух пальцах вверх
- [ ] Ложные срабатывания минимальны

### Стабильность
- [ ] Система работает 30+ минут без перезагрузки
- [ ] Нет утечек памяти
- [ ] Нет визуальных артефактов
- [ ] Статистика обновляется корректно

### Удобство
- [ ] Управление курсором интуитивно
- [ ] Клики срабатывают когда надо
- [ ] Визуальная обратная связь четкая
- [ ] Интерфейс отзывчив

## Примеры использования в реальных сценариях

### Сценарий 1: Управление презентацией
```javascript
// В app.js добавьте:
if (gesture === 'POINT') {
    // Переход к следующему слайду
    presentationController.nextSlide();
}
if (gesture === 'VICTORY') {
    // Переход к предыдущему слайду
    presentationController.previousSlide();
}
```

### Сценарий 2: Рисование
```javascript
// Отслеживание движения ладони для рисования
if (gesture === 'PALM' || gesture === 'OPEN_HAND') {
    drawingCanvas.drawLine(lastPos, currentPos);
}
if (gesture === 'FIST') {
    // Стереть линию
    drawingCanvas.clear();
}
```

### Сценарий 3: Игра
```javascript
// Использование жестов как команды
if (gesture === 'VICTORY') {
    game.jump();
}
if (handY > prevY) {
    game.moveDown();
}
if (handX < prevX) {
    game.moveLeft();
}
```

### Сценарий 4: Медиаплеер
```javascript
// Управление видео через жесты
if (gesture === 'THUMBS_UP') {
    videoPlayer.play();
}
if (gesture === 'FIST') {
    videoPlayer.pause();
}
if (palmRight) {
    videoPlayer.forward(10);
}
```

## Отладка в консоли браузера

### Включить логирование
В `app.js` добавьте в `onResults()`:
```javascript
console.log('Hands detected:', this.handData.hands.length);
console.log('Gestures:', Array.from(this.handData.gestures.values()));
console.log('Cursor pos:', this.smoothedHandPos);
```

### Проверить конфигурацию
В консоли браузера (F12):
```javascript
// Просмотреть текущую конфигурацию
console.log(gestureControl.config);

// Изменить конфигурацию динамически
gestureControl.config.minConfidence = 0.3;
```

### Профилирование производительности
```javascript
console.time('gesture_recognition');
// ... код ...
console.timeEnd('gesture_recognition');
```

## Известные ограничения

### Оборудование
- Требуется вебкамера (встроенная или внешняя)
- Требуется интернет для загрузки MediaPipe (или сохранить локально)
- Требуется мощность процессора для обработки видео

### Программное обеспечение
- Требуется современный браузер с WebRTC
- Работает в браузере, не как отдельное приложение
- Требуется JavaScript включен

### Условия
- Требуется хорошее освещение
- Требуется чистая область перед камерой
- Сложнее распознает закрытые пальцы за другими предметами

## Оптимизация для вашего оборудования

### Если медленно:
```javascript
// В app.js уменьшить сложность модели
modelComplexity: 0, // Было 1 - легче для процессора
```

### Если часто ошибается:
```javascript
// Установить более высокий порог
minDetectionConfidence: 0.7 // Было 0.5
```

### Если дергается курсор:
```javascript
// В приложении увеличить сглаживание до 80-90%
// Или в config.js:
CURSOR_SMOOTHING: 0.9
```
