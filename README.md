# 🖐️ Hand Gesture Control System

Современная веб-система управления интерфейсом через жесты рук с использованием MediaPipe и TensorFlow.js. Распознает движения рук в реальном времени для бесконтактного управления.

## ✨ Особенности

- 🎯 **Распознавание жестов в реальном времени** - детектирует 21 точку на каждой руке
- 👊 **Бесконтактное управление** - сжатие кулака = клик, движение ладони = управление курсором
- 🎨 **Современный темный дизайн** - красивый интерфейс с эффектами и анимациями
- 📊 **Подробная статистика** - отслеживание уверенности, количества рук, текущих жестов
- ⚙️ **Настраиваемые параметры** - регулировка чувствительности и сглаживания
- 📱 **Адаптивный дизайн** - работает на разных размерах экранов
- 🔄 **Плавное управление** - алгоритм сглаживания для точного управления курсором

## 🎮 Поддерживаемые жесты

| Жест | Описание | Действие |
|------|---------|---------|
| 👊 | Сжатый кулак | Клик |
| ✋ | Открытая ладонь | Управление курсором |
| 🖐️ | Раскрытая рука | Управление курсором |
| 🤟 | Знак V (victory) | Дополнительное действие |
| ☝️ | Указание одним пальцем | Точное указание |
| 👍 | Палец вверх | Специальный жест |

## 🚀 Быстрый старт

### 1. Откройте файл в браузере
Просто откройте `index.html` в современном браузере, который поддерживает:
- WebRTC API (доступ к камере)
- WebGL (для обработки видео)
- Web Workers (для MediaPipe)

### 2. Разрешите доступ к камере
При первом запуске браузер попросит разрешение на доступ к вебкамере. Нажмите "Разрешить".

### 3. Нажмите "Запустить"
Кнопка ▶️ запустит систему распознавания жестов.

## 🎯 Использование

### Управление курсором
1. Откройте ладонь перед камерой
2. Кончик указательного пальца будет следовать за курсором
3. Курсор плавно следует благодаря алгоритму сглаживания

### Клик
1. Покажите открытую ладонь
2. Сожмите кулак (закройте все пальцы)
3. Вы услышите/увидите обратную связь о клике
4. Элемент, указанный курсором, будет нажат

### Настройка параметров
- **Уверенность обнаружения** - минимальный порог уверенности для обнаружения рук
- **Сглаживание** - сила сглаживания движения курсора (0 = без сглаживания, 1 = максимум)
- **Зеркальное отображение** - отражение видео по горизонтали
- **Показывать скелет** - показывать точки и соединения рук

```javascript
HandGestureControl

├── Обработка видеопотока
├── Распознавание жестов
├── Обработка действий (клики, движение)
└── Обновление UI
```

### Алгоритм распознавания жестов

1. **Получение ориентиров (landmarks)** - 21 точка на руке
2. **Анализ положения пальцев** - какие пальцы подниманы
3. **Расчет открытости руки** - процент раскрытия
4. **Классификация** - определение типа жеста
5. **Выполнение действия** - управление курсором или клик

## 🐛 Отладка

### Проблемы с камерой
1. Проверьте разрешение браузера на доступ к камере
2. Убедитесь, что другие приложения не используют камеру
3. Перезагрузите страницу

### Плохое распознавание жестов
1. Увеличьте освещение
2. Уменьшите значение "Уверенность обнаружения"
3. Приближайте/отдаляйте руку до оптимального расстояния (30-70 см)

### Медленное управление курсором
1. Увеличьте значение "Сглаживания"
2. Закройте другие тяжелые приложения
3. Проверьте производительность браузера
