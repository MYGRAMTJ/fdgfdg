# 🎯 Hand Gesture Control System - Complete Package

## 📦 Что входит в пакет

Полнофункциональная система управления веб-интерфейсом через жесты рук с использованием MediaPipe и TensorFlow.js.


## ✨ Ключевые особенности

### 🎮 Распознавание жестов
- ✅ **6+ типов жестов** - кулак, ладонь, V-знак, указание, палец вверх
- ✅ **Двойная рука** - одновременное распознавание двух рук
- ✅ **Реальное время** - обработка в реальном времени
- ✅ **ML модель** - MediaPipe Hands с 21 точкой на руке

### 🖱️ Управление интерфейсом
- ✅ **Управление курсором** - движение ладони = движение курсора
- ✅ **Клик** - сжатие кулака = клик мышью
- ✅ **Плавное движение** - алгоритм сглаживания
- ✅ **Виртуальный курсор** - визуализация позиции

### 🎨 Дизайн

- ✅ **Адаптивный** - работает на всех размерах экранов

## 🎯 Общие сценарии использования

### 1️⃣ Презентации
```javascript
// Сжать кулак → следующий слайд
// Открыть руку → вернуться доступ к курсору
```
Пример в: `TESTING.md` → Сценарий 1

### 2️⃣ Рисование по воздуху
```javascript
// Открытая ладонь → включить рисование
// Сжатый кулак → очистить холст
```
Пример класса: `AirDrawing` в `examples.js`

### 3️⃣ Управление музыкой/видео
```javascript
// 👍 Палец вверх → включить воспроизведение
// 👊 Кулак → пауза
// 🤟 V-знак → вперед на 10 сек
```
Пример класса: `GestureMediaController` в `examples.js`

### 4️⃣ Игры с жестовым управлением
```javascript
// Движение ладони влево-вправо → управление персонажем
// V-знак → прыжок
// Палец вверх → быстрый бег
```
Пример класса: `GestureGameController` в `examples.js`

### 5️⃣ Навигация по меню
```javascript
// V-знак → вверх в меню
// Знак вверх → вниз в меню
// Кулак → выбор
```
Пример класса: `GestureMenuNavigator` в `examples.js`

- Если ошибается: увеличить `minDetectionConfidence`
- Если дергается: увеличить `smoothing` в приложении

## 🐛 Диагностика проблем

### Проблема: "Камера не работает"
```
1. Проверить разрешение браузера на камеру
2. Открыть консоль (F12) на наличие ошибок
3. Перезагрузить страницу
4. Попробовать другой браузер
```

### Проблема: "Рука не распознается"
```
1. Увеличить освещение
2. Уменьшить порог уверенности до 30%
3. Показать рук полностью в центре экрана
4. Отстаньте на расстояние 30-70 см
```

### Проблема: "Курсор дергается"
```
1. Увеличить значение сглаживания до 80%
2. Закрыть фоновые приложения
3. Обновить страницу
4. Перезагрузить браузер
```
