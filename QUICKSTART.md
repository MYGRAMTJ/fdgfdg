# 🚀 Быстрый старт - Hand Gesture Control

## За 5 минут до первого использования

### ШАГ 1: Подготовка (30 секунд)
1. Убедитесь, что у вас есть веб-камера
2. Закройте другие приложения, использующие камеру
3. Окружение хорошо освещено (сбоку или спереди лучше)

### ШАГ 2: Открытие приложения (30 секунд)
```
Просто откройте файл index.html в браузере:
- Двойной клик на index.html
- ИЛИ перетащите файл в браузер
- ИЛИ используйте контекстное меню "Открыть с..."
```

**Рекомендуемые браузеры:**
- Chrome 90+ ✅
- Firefox 88+ ✅
- Edge 90+ ✅
- Safari 14.1+ ✅

### ШАГ 3: Разрешение доступа (30 секунд)
```
Когда браузер спросит разрешение на доступ к камере:
1. Нажмите на иконку камеры в адресной строке
2. Выберите "Разрешить"
3. При необходимости выберите нужную камеру
```

### ШАГ 4: Запуск (30 секунд)
```
В интерфейсе приложения:
1. Нажмите кнопку "▶️ Запустить"
2. Дождитесь появления видео с вашего лица
3. Откройте одну из рук перед камерой
```

### ШАГ 5: Первая попытка (3 минуты)
```
1. Откройте ладонь перед камерой
   → Вы должны увидеть скелет руки
   → Виртуальный курсор будет следовать за пальцем

2. Медленно воздействуйте рукой
   → Курсор должен плавно двигаться

3. Сожмите кулак (все пальцы закрыты)
   → Должна появиться анимация клика (💥)
   → Статус должен измениться на "Кулак (Клик)"

4. Откройте руку снова
   → Курсор вернется под контроль
```

## 🎯 Минимальная конфигурация

Если жесты не распознаются:

1. **Уменьшить порог уверенности:**
   - В panel справа сдвиньте "Уверенность обнаружения" на 30-40%

2. **Добавить освещение:**
   - Включите люстру/настольную лампу
   - Сядьте лицом к свету

3. **Отрегулировать расстояние:**
   - Держите руку на расстоянии 30-70 см от камеры
   - Рука должна быть полностью видна

## 🎓 Первые упражнения

### Упражнение 1: Управление курсором (2 мин)
```
1. Откройте руку (все пальцы вверх)
2. Медленно двигайте рукой в разные стороны
3. Обратите внимание на плавность движения курсора
4. Если дергается - увеличьте "Сглаживание" на 10%
```

### Упражнение 2: Клик (2 мин)
```
1. Поднесите палец к элементу (кнопке)
2. Полностью сожмите руку в кулак
3. Услышьте/увидьте эффект клика (💥)
4. Откройте руку - был ли выполнен клик?
```

### Упражнение 3: Точность (2 мин)
```
1. Наведите курсор на кнопку "🔄 Сброс"
2. Сожмите кулак
3. Она должна была сработать
4. Повторите 5 раз для тренировки
```

## 🔊 Включение звука (опционально)

Чтобы услышать звуки жестов, отредактируйте в `app.js`:
```javascript
this.audioEnabled = true; // Меняем на true
```

## 📸 Советы для лучшего распознавания

### Освещение
```
✅ Хорошее:    Окно сбоку + настольная лампа
✅ Хорошее:    Яркое потолочное освещение
❌ Плохое:     Только экран компьютера

### Расстояние
```
✅ Оптимально:  30-70 см от камеры
⚠️ Минимум:     20 см
⚠️ Максимум:    120 см
```

### Фон
```
✅ Хорошо:     Однотонный светлый фон
✅ Хорошо:     Контрастный фон
❌ Плохо:      Много движущихся объектов
❌ Плохо:      Много похожих на шум паттернов

### Положение руки
```
✅ Хорошо:     Рука полностью видна в кадре
✅ Хорошо:     Пальцы ясно видны
❌ Плохо:      Рука обрезана краями экрана
❌ Плохо:      Пальцы скрыты
```

## 🐛 Решение проблем

### "Камера не работает"
```
1. Проверьте в Параметры системы > Приватность > Камера
2. Убедитесь, что браузер имеет разрешение
3. Перезагрузите браузер
4. Попробуйте другой браузер
```

### "Рука не распознается"
```
1. Уменьшить "Уверенность обнаружения" на 20%
2. Включить больше света
3. Отдалить/приблизить руку
4. Открыть рук полностью (не сгибать пальцы)
5. Переместить рук в центр экрана
```

### "Курсор дергается"
```
1. Увеличить "Сглаживание" с 60% до 80-90%
2. Закрыть другие тяжелые приложения
3. Обновить браузер
4. Проверить процессор (может быть перегружен)
```

### "Клик не срабатывает"
```
1. Убедиться, что статус показывает "Кулак (Клик)"
2. Зажать кулак на 1 секунду
3. Полностью закрыть все пальцы (включая большой)
4. Увеличить "Уверенность обнаружения" если часто ошибается
```

## 🎨 Первая кастомизация

### Изменить цвет акцента
В файле `styles.css` найдите:
```css
:root {
    --accent: #00d4ff; /* Измените на свой цвет */
}
```

Примеры цветов:
- `#00ff00` - зеленый
- `#ff0000` - красный
- `#ffff00` - желтый
- `#ff00ff` - фиолетовый

### Изменить название
В файле `index.html` найдите:
```html
<h1 class="logo">✋ Hand Control</h1>
<!-- Измените текст -->
```

### Отключить звуки
В файле `app.js` найдите:
```javascript
this.audioEnabled = false; // Отключит звуки
```
